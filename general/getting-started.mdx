# Getting Started

## Installation

Getting started with **Fonk** is something really easy and quick. First of all you will need to install **Fonk** in your existing project using your
favorite package manager (we'll assume npm for this example).

```bash
npm install @lemoncode/fonk --save
```

> **Fonk** is Typescript friendly, typing definitions are already included in the main package.
